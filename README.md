# Java2DGameEngine

A simple 2d game engine made with Java

## Requirements

Java 8 JRE or JDK

## QuickStart

- Download the [latest release](https://github.com/BertrandAlexandre/Java2DGameEngine/releases).
- Import `Java2DGameEngine.jar` into your project dependencies.
- Use.
